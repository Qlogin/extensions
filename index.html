Password Manager v0.3

New most powerful tool to control your passwords.
Using only one confidential password or moto the Password Manager generates the unique password for each site that you use. It does not keep your data or generated passwords anywhere, so it won't be intercepted by malefactors.
This extension based on idea and code of Parole Manager.
